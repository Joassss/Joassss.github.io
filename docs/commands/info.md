# Info Commands

> Last edited: 1/22/2022 at 11:54PM

Info commands are commands used to get statistics, information and loads of other stuff.

#### `/announcement`
Latest quabot news.

#### `/help`
List commands and their descriptions.

#### `/info`
Information about the bot.

#### `/leaderboard`
Server XP's leadersboard.

#### `/nick [nickname] (user)`
Change the nickname of a user.

#### Example
- `/nick John`
- `/nick Edward @Wobbe_`

#### `/online`
View activity list.

#### `/ping`
Get bot ping.

#### `/rank [user]`
Get a user's XP and rank.

#### Example
- `/rank @Joa_sss`

#### `/roles`
List of roles in this guild.

#### `/serverinfo`
erver information.

#### `/stats`
Bot statistics.

#### `/support`
Bot's support discord.

#### `/uptime`
Bot's uptime.

#### `/userinfo (user)`
Information about a user.

#### Example
- `/userinfo`
- `/userinfo @Joa_sss`
